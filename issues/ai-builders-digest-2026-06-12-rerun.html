Loopcraft, Infrastructure & Commerce / Loop 工艺、基础设施与商业

swyx 论 loop 堆叠、Rauchg 展示 Vercel+Shopify、Steipete 让 Codex 做 PR。

S
Swyx
Writer / Builder
@swyx
中文

swyx 的 Loopcraft 宣言：「下一个世纪的游戏就是尽可能高效地堆叠 loop。」在早期，知道什么时候往下钻进 loop（为了可靠性）很有价值。但更有价值的是知道随着模型改进怎么往上走 loop（为了杠杆）。「如果你不想明白这个，别嫉妒那些想明白了的人赢你。」29 赞。

这是一个思考 AI 辅助开发中抽象层的框架：每一层是一个自动化工作的「loop」。技能在于知道什么时候深入 loop 做调试，什么时候随模型改进上升到更高层获取杠杆。

swyx Loopcraft：堆叠 loop 是下世纪的游戏。往下钻为可靠，往上去为杠杆。29 赞。

English

swyx's Loopcraft manifesto: 'The entire game of the next century is to be able to stack loops as effectively as possible.' In early days, it's valuable to know when to go DOWN a loop when things go wrong (for reliability). But it's more valuable to know how to go UP a loop as models improve (for leverage). 'If you don't figure out how to do this, don't be salty when you lose to those that do.' 29 likes.

This is a framework for thinking about abstraction layers in AI-assisted development: each layer is a 'loop' of automated work. The skill is knowing when to dive deeper into a loop for debugging and when to rise up for more leverage as models improve.

On Loopcraft: One might argue the entire game of the next century is to be able to stack loops as effectively as possible. In the early days of each phase, it will be valuable to know when to go DOWN a loop when things go wrong (for reliability)… but it will probably be more valuable to know how to go UP a loop as models improve (for leverage). If you don't figure out how to do this, don't be salty when you lose to those that do.

Podcast: Logan Kilpatrick — The Model Eats the Harness / 播客：Logan Kilpatrick — 模型吃掉 Harness

Google DeepMind 的 Logan Kilpatrick（Google AI Studio / Gemini API 负责人）与 Training Data 对谈：Antigravity 作为 Google 的 agent through line、模型吃掉 harness 的必然性、Omni 世界模型、DeepMind 文化。

TD
Training Data
Sequoia Podcast
中文

Google AI Studio / Gemini API 负责人 Logan Kilpatrick 和 Training Data 对谈 Google 的 agent 时代。核心洞察：（1）Antigravity 是 Google 新的贯通线——不只是 IDE，是一个生态系统（IDE、web、CLI、SDK），驱动 Search、Gemini app、Cloud 和 AI Studio 中的 agent。（2）关于 coding 竞争：承认 Gemini 在 coding 心智份额上落后，但说 3.5 Flash 纯靠 post-training 就超过了之前所有 Pro 模型的 coding 能力。（3）「模型吃掉 harness」——我们现在叫「模型」的东西已经不只是权重了，是一个包含工具、容器、agent 基础设施的扩张系统。脚手架会被吸收进原生模型，大约 12 个月一个周期。（4）Omni 是真正的单一模型世界模型：不是路由到不同子模型，是一个模型处理文本、音频、视频、图片编辑。（5）DeepMind 文化：Demis 是诺贝尔奖科学家，Sam 是最好的商人之一，Dario 是神秘主义者——你能感觉到各自公司的 DNA。（6）Coding 的窄超级智能已经到了：它是加速器不是替代品，让人类开发者更有能动性。

Kilpatrick：Gemini 之前 Google 产品之间没有贯通线。Antigravity 成了新的贯通线。

12 月大家说 Google 赢了，然后 agentic coding 出来叙事变得很快。提醒一切变得多快。

模型吃掉 harness。我们叫模型的已经不只是权重了，是扩张的系统。脚手架比模型先走两步，然后被模型吃掉。

Omni 是真正的单一模型，不是路由到不同模型。是新架构让它成为可能。

Demis 是诺贝尔科学家，Sam 是最好的商人，Dario 神秘。各公司 DNA 不同。

Coding 的窄超级智能已经到了。没有夺走人类开发者的价值，是加速器。我感觉自己更有能动性。

Antigravity 不只是 IDE：是 IDE、web、CLI、SDK 的生态系统。同一个 harness 驱动 Search、Gemini app、Cloud 的 agent。

3.5 Flash 纯靠 post-training 就超过了之前所有 Pro 模型的 coding 能力。

需要 harness bench 来衡量模型对不同 harness 的适应能力。

Omni 第一个发布的能力是视频编辑，因为最成熟。未来版本会更强大。

AI Studio 里 20% 的应用是游戏，金融和加密货币相关也占 20%。Android 成为 builder 的平台。

AI 采纳最多公司反而人头增长最多。AI 不消灭工作，而是让公司启动更多项目。

专注是初创公司的超能力。大公司做不到专注，这就是创业机会所在。

DeepMind 的文化：一群聪明人聚在一起解决问题。科学导向，不只是追 benchmark。

我们应该衡量 agent 平均运行时长——这个指标在快速上升。

English

Logan Kilpatrick (Google AI Studio / Gemini API lead) talks with Training Data about the agentic era at Google. Key themes: (1) Antigravity is the new through line for Google—it's not just an IDE, it's an ecosystem (IDE, web, CLI, SDK) that powers agents across Search, Gemini app, Cloud, and AI Studio. The same harness that codes is the harness that runs agents everywhere. (2) On coding competition: 'December, the narrative was Google had won. Then the agentic coding stuff happened and the narrative shifted fast.' He acknowledges Gemini is behind on coding mindshare but says the 'avengers of AI' team is pushing hard, and 3.5 Flash surpassed every previous Pro model on coding using only post-training gains. (3) 'The model eats the harness'—what we call 'the model' today is already not just weights; it's an expanding system of tools, containers, and agent infrastructure. The scaffolding gets absorbed into the native model system over 12 months. He calls for a 'harness bench' to measure how models adapt to different harnesses. (4) Omni as a true single-model world model: not routing to different sub-models, but one model that handles text, audio, video, image editing. The video editing use case is the first to ship because it's the most mature. (5) On DeepMind culture: 'Demis is a Nobel Prize scientist, Sam is one of the world's best businessmen, Dario is esoteric—you feel that in each company's DNA.' The scientific approach at DeepMind keeps them focused on solving real human problems, not just pushing benchmark numbers. (6) Narrow superintelligence in coding is already here: 'It feels like it hasn't taken away from human developers. It's an accelerant. I feel like I have more agency.'

Historically, prior to Gemini, there actually wasn't a through line for the probably sub 100 number of Google products. Gemini became this through line. Now Antigravity is becoming the new through line.

December, the narrative was that Google had won. Then the next wind of the narrative was all the agentic coding stuff. It's a reminder of just how fast things can change.

The model eats the harness. What we call the model is not just weights anymore. It's an entire expanding sprawling system. The scaffolding is oftentimes a couple of steps ahead of what is baked directly into the model, and then the model eats that scaffolding.

Omni is a true single model, not routing to different models. It's a new setup that makes that possible.

Demis is a Nobel Prize scientist. Sam is one of the world's best businessmen. Dario is esoteric. You feel that in the DNA of each company.

It feels like narrow superintelligence for coding. It hasn't taken away from human developers. I feel like I have more agency in the world.
